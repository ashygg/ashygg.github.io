At the time that I began to work on my second Python project, I had improved my basic skills in comparison to when I had started my first project. As such, I proposed starting a more challenging project, which was to analyze my personal Spotify streaming data. To do so, requesting the data in advance was necessary. And because this project was more advanced/complex and utilized modules that I had never implemented before, some online help was definitely necessary. I had decided to analyze one file of my streaming history (out of 4) and find my 'most listened to song' and 'most listened to artist' of that time period. Because I had guidance, it didn't seem too bad, but I don't think I would've been able to produce the code without it. But, I am proud that I was able to figure out a way to extract the data that I wanted to see and personalize the code to do so.
